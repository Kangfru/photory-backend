# 영화 포토티켓 어플리케이션 백엔드

## 소개

이 백엔드 어플리케이션은 영화 포토티켓 어플리케이션의 기능을 지원하기 위해 설계되었습니다. 이 프로젝트의 핵심 목표는 TheMovieDB API를 통해 검색된 영화 또는 OTT 데이터를 사용하여 포토티켓을 생성하고 관리하는 것입니다.

어플리케이션의 사용자들은 포토티켓을 생성함으로써 '도장깨기'를 경험하고 내역을 관리할 수 있습니다.

## 기술 스택

- Java SDK 버전 11
- Spring Boot 2.7.2
- MongoDB 3.x
- Lombok
